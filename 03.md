# BAT产品人生存法则-沟通之道 #
## Live 简介 ##
我是舒文，腾讯产品经理，前百度商业产品经理，新浪读书／当当签约作者，自由撰稿人。

求职面试、商务谈判、日常沟通、述职晋级、公开演讲、升迁加薪、PK ，无时无刻离不开说话。人人都能说话，但是不是每个人都会说话。说话的方式方法、语境技巧直接关系到一次谈判的结果。产品经理是一个对从业者综合能力要求很高的职业，谈判沟通更是 BAT 这类公司考核产品经理时看重的关键素质。

产品经理的日常工作离不开与开发、设计、商务、运营沟以及跨部门、跨公司合作的谈判沟通。对于沟通经验丰富的人，一次简短的谈话立刻分辨你是刚入职场的小白，还是久经沙场的职场达人。

想晋级高阶产品经理，扩大个人影响力，需要不仅是会说漂亮话。恰如其分地抬杠，熨贴至里地关心更显得尤为重要。我将在本次 Live 中总结在百度、腾讯从业期间的沟通实战经验技巧，希望通过收听本 Live 能够帮助大家更高级、更高效地沟通。

本次 Live 主要包括以下内容

- 学会闭嘴，进入新环境请先用心观察
- 调整心态，语速语气可以帮助你提升气场
- 向上汇报，做到以下几点就够了
- 如何游刃有余地和你的 peers 抬杠
- 不当受气包，发火要选对时机
- 跨部门合作，请先不要说你想要什么
- 自营销，学会使用高级的演讲述职技巧
- 一些个人见解
