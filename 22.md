# 如何自学 Java ：入门与编程技能训练 #
## Live 简介
金旭亮，二十四年前，我开始自学计算机，后到北京读研并留校任教，一直以来，北京理工大学计算机学院本科生的 Java 课都是我讲的，颇受欢迎，己历时超过十年，对于初学者学习情况比较了解，同时自己也是一名老程序员，参加或主持过不少软件项目，用过一打以上的编程语言或特定技术写程序，独立开发了自己的计算机教学网站，并写了 6 本软件技术书籍和教材。

本次 Live 专为 Java 自学者定制，明确列出了要学习的技术点，给出了详细的学习建议和具体路线，介绍了 Java 开发技能的科学训练方法。 

本次 Live 的听众可以对 Java 一无所知，但最好有过一门语言（最好是 C ）的编程经验，对那些学习时间充裕的理工科专业（信息类相关专业尤佳）低年级大学生尤其合适。

适合的另一批人，就是想跳槽到 IT 行业，不能或不愿上培训班的在职人员，他们选择自学 Java 很合适，因为 Java 就业机会之多名列前茅。

立即要找工作的人不必听，这场 Live 所介绍的学习路线过于「漫长」，没办法，专业技能的培养需要时间和耐心。 

零编程基础的人只能听懂一小部分，事实上，我确信几乎所有听众一定会有部分内容听不懂，因为本次 Live 的目的是勾勒出一个完整的学习路线，因此必然会涉及超出听众当前所了解的知识领域的部分。 

最后补充一下：随着自学的具体进程，本次 Live 中的相应部分，需要多次反复听。

课程地址：[https://www.zhihu.com/lives/889495940065538048/](https://www.zhihu.com/lives/889495940065538048/)

本次 Live 主要包括以下内容

- 学习 Java 要完成的主要任务是什么？
- 初学者应该按照什么样的顺序去学 Java ？
- 哪些技术与技能极为重要，必须下苦功夫掌握？
- 有哪些技术「现在必须要学」和可以「以后再学」？
- 有哪些技术己经过时可以不必再学？
- 特定的技术（比如 JDBC ， Socket 开发、多线程……）应该怎样学？
- 怎样学习与掌握比较复杂的 Java 开发框架（比如 Spring Framework 和 Netty ）？
- Java 技术领域近些年有哪些新技术是需要关注并掌握的？
- 怎样自己找到合适的项目练手？如何通过刻意训练提升自己的开发能力？
- ……

## Live笔记
	各位朋友，大家好，非常感谢大家对我的信任，来选择参加我主持的Live!
	现在离Live开讲还有较长的时间，请参加Live的朋友先按照自己的计划自学，对Java零基础的，可以先看一些Java入门级别的书籍或视频，这里推荐三本书籍（附上京东链接）：
	（1）英文好的，可以看《Java How To Program》Deitel著，https://item.jd.com/12111589.html
	（2）中文书籍，李刚的《疯狂Java讲义》，https://item.jd.com/11514816.html
	（3）O'Reilly出的《Head First Java（中文版）》，https://item.jd.com/11514816.html
	任选一本即可。
	对Java先有个基本了解，有助于听懂Live中介绍的内容，并从中获取最大的收益。
	Live重点是介绍Java技术学习路线、自学建议和编程技能的自我刻意训练方法，并且会将Java入门阶段最重要的必须要掌握的重点技术，以通俗易懂的方式，以尽可能简洁（甚至就用一句话、一个比喻）来点出其中的关键，涉及面比较广，同时介绍的难度也并不浅，因此，对于初学者来说，信息量可能会超过你当前所能消化的上限，在此先打个预防针。
	建议随着自学的进度，有针对性地在日后重温特定的部分。比如我在Live中会介绍当下流行的高性能网络开发框架Netty的学习要点，对于初学者来说，现在肯定是听不懂的，但等你按照我给出的学习路线，扫清了前面的障碍，具备学习Netty这项技术的前提时，先来复习一下Live中对它的介绍，然后再去看Netty的技术书籍或学习资源，就能获得较高的学习效率，得到较好的学习效果。
	另外，大家在自学Java过程中最困惑的地方在哪，请发在本次Live中，为了避免问题重复，请大家在发问前最好先看看有没有其他人己经发过类似的问题，有的，直接点击“喜欢”按钮，这样我就知道哪些问题大家最关心，从而在Live中有针对性地给以解答。
	还有，为了避免无关的信息泛滥，请尽量只发有意义的文字，这样能有效地节约所有人的时间。
	参加这种Live的朋友，都是爱好学习的，让我们一起学习，共同进步！

	我看了大家提的问题，不太适合放入Live的内容，我写成了专栏文章给出回答。文章链接如下：http://weibo.com/ttarticle/p/show?id=2309404157053751951278，这是发在新浪微博的，知乎专栏文章链接为：https://zhuanlan.zhihu.com/p/29761681，推荐访问新浪微博上的文章，因为知乎上的文章有错别字，又不支持设置颜色，新浪的头条文章看起更舒服。
	更多的问题请继续，期望本次Live能给大家以切切实实的帮助。
